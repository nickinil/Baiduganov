# Weather Diary (Дневник погоды)

**Автор:** Байдуганов Михаил Владиславович

## Краткое описание
Приложение с графическим интерфейсом (Tkinter) для ведения дневника погодных наблюдений. Позволяет:

- Добавлять записи с датой, температурой, описанием и признаком осадков.
- Просматривать все записи в удобной таблице.
- Фильтровать записи по дате и/или минимальной температуре.
- Сохранять данные в файл JSON и загружать их обратно.
- Проверять корректность вводимых значений.

## Как запустить приложение

1. Убедитесь, что установлен Python версии 3.6 или выше.
2. Скопируйте файлы проекта в отдельную папку:
   - `MAIN.py` – основная программа
   - `weather_data.json` – файл с данными (создаётся автоматически)
   - `README.md` – этот файл
   - `tests.md` – сценарии тестирования
3. Запустите приложение из командной строки:
   ```bash
   python MAIN.py
